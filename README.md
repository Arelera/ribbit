# ribbit

[Visit Site](https://theribbit.herokuapp.com/)

Ribbit is a reddit clone built with the PERN stack.

## Used technologies

### Frontend

- ReactJS
- Redux, react-redux, redux-thunk
- Styled Components
- Axios
- Date-fns

### Backend

- Express
- Pg (node-postgres)
- Bcryptjs
- Jsonwebtoken
